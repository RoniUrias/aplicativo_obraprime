 Obra Prime 

> Aplicativo completo para gestão de obras civis, controle de Ordens de Serviço (OS) e agendamento de equipes.

 Sobre o Projeto

O Obra Prime é uma solução desenvolvida para facilitar o dia a dia no canteiro de obras e no escritório. O objetivo é conectar administradores, equipes de campo e clientes, permitindo o acompanhamento em tempo real do progresso das obras.

A interface foi projetada no **Figma** com foco em usabilidade (UX) para garantir que o preenchimento de relatórios e OS seja rápido e intuitivo.

---

Funcionalidades Principais

O app conta com os seguintes módulos:

Autenticação Segura: Telas de Login e Cadastro (Inicial, Segunda, Terceira).
Dashboard (Home): Visão geral das obras em andamento e notificações.
Agenda Inteligente: Organização de visitas técnicas e cronograma de obras (Agenda 1-4).
Gestão de O.S.: Criação, edição e acompanhamento de Ordens de Serviço.
Perfis de Acesso: Diferenciação entre Administrador e Usuário comum.
Painel Admin: Controle total de usuários e configurações do sistema.

Back-end

Como rodar o projeto localmente

Acessar a pasta do projeto e executar o comando "npm start" para iniciar o front 
Acessar a pasta back end dentro do projeto e executar o comando "node server.js"

Pré-requisitos

Node.js instalado (versão LTS 18 ou 20).

Expo CLI instalada com:
npm install -g expo-cli

Git instalado (opcional, mas recomendado).

Editor VS Code com extensões básicas (React, Prettier).

App Expo Go no celular para escanear o QR Code.

Instalar dependências do projeto usando:
npm install
